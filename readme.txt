1. In this directory, open up the console and type "gradlew run"
2. You will be at the menu screen at this point. Press enter to start the game
3. Use arrow keys OR WASD to move the character around. Try to survive all the bullets for the highest score. You can press 'P' to pause the game.
4. When you get hit you will move to the game over screen. Press ENTER to restart the game or press 'T' to go back to the main menu.

** Pressing ESC at any time will exit the game.
